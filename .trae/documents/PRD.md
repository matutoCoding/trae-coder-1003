## 1. 产品概述

林业碳汇项目管理Web系统，为林场提供林地管理、碳汇监测、计量核算和碳交易全流程管理，助力实现碳达峰碳中和目标。

- 主要面向林场管理人员、碳汇项目开发者、第三方核查机构及交易参与方
- 解决碳汇项目管理分散、数据孤岛、核算流程复杂等问题
- 实现林地资源数字化、碳汇计量自动化、交易流程规范化

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 系统创建 | 用户管理、权限配置、系统设置 |
| 林场管理员 | 账号注册 | 项目台账、林地资源、碳汇监测、计量核算、营林管护、收益分配 |
| 核查人员 | 账号注册 | 碳汇项目核查、数据审核、报告出具 |
| 交易人员 | 账号注册 | 交易管理、碳汇签发、碳排放权交易 |

### 2.2 功能模块

1. **项目台账**：碳汇项目档案、项目年度报告
2. **林地资源**：林地小班区划、林权权属管理
3. **碳汇监测**：树种胸径测量、固碳增量监测
4. **计量核算**：生物量碳储量核算、碳汇计量模型
5. **交易管理**：第三方核查、碳汇签发登记、碳排放权交易
6. **营林管护**：营林抚育管护、抚育作业记录
7. **收益分配**：收益分配台账、收益统计分析

## 3. 核心流程

```mermaid
flowchart LR
    A["项目立项登记"] --> B["林地资源调查"]
    B --> C["小班区划确权"]
    C --> D["碳汇基线设定"]
    D --> E["定期监测计量"]
    E --> F["生物量核算"]
    F --> G["第三方核查"]
    G --> H["碳汇量签发"]
    H --> I["碳排放权交易"]
    I --> J["收益分配"]
    J --> K["年度报告"]
```

项目从立项登记开始，经过林地调查、小班区划、基线设定后进入定期监测，通过生物量核算和第三方核查后进行碳汇签发，最终完成交易和收益分配，每年形成年度报告闭环。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：森林绿 (#2D5A27)，象征自然、生态、可持续发展
- **辅助色**：大地棕 (#8B6914)、天空蓝 (#4A90A4)、金色 (#D4A84B)
- **背景色**：浅米色 (#F5F1E8)，营造自然舒适的视觉感受
- **按钮风格**：圆角矩形，微立体效果，主按钮采用森林绿渐变
- **字体**：标题使用"Noto Serif SC"衬线字体，正文使用"Noto Sans SC"无衬线字体
- **布局风格**：左侧导航栏 + 顶部操作栏 + 右侧内容区的经典管理后台布局，采用卡片式模块划分
- **图标风格**：线性图标配合填充色块，使用绿色系图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|---------|
| 项目台账 | 碳汇项目档案 | 数据表格、状态标签、筛选器、详情弹窗、操作按钮组 |
| 林地资源 | 林地小班区划 | 地图组件、区划多边形、信息面板、数据统计卡片 |
| 碳汇监测 | 固碳增量监测 | 折线图、柱状图、数据看板、监测点标记 |
| 计量核算 | 生物量碳储量核算 | 公式编辑器、计算参数表、结果展示区、导出按钮 |
| 交易管理 | 碳排放权交易 | 交易流水表、合同预览、状态进度条、金额统计 |
| 营林管护 | 营林抚育管护 | 日历组件、任务卡片、进度圆环、人员分配表 |
| 收益分配 | 收益分配台账 | 分配明细表、饼图分析、到账状态标记、汇总报表 |

### 4.3 响应式设计

- **桌面优先**：以1920px为基准设计，支持1440px、1280px适配
- **平板适配**：最小支持1024px宽度，导航栏可收起
- **交互优化**：表格支持横向滚动，表单元素触控区域不小于44px
- **图表自适应**：ECharts图表随容器宽度自动重绘

### 4.4 动效设计

- 页面加载采用渐入 + 上移动画，分模块依次展示（stagger效果）
- 数据表格行hover时背景色微变 + 轻微放大
- 侧边导航展开/收起采用平滑过渡动画
- 数字统计变化采用滚动计数动效
- 图表数据采用渐进式渲染动画
